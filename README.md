# Translucent GUI

This resourcepack adds translucency to the Minecraft GUI, which was previously not possible in vanilla for versions 1.21.1 and below. To add support for older versions of the game, a pixel perfect combination of black and transparent pixels was used. This results in a seemingly translucent look. Newer versions have luckily added translucency support making this approach obsolete.

The goal of this pack was to keep the vanilla look of the game but add transparency so the interfaces are less distracting when playing.
